# 🥜 Peanut

**Peanut** is a **quest-focused, gamified mobile app** that lets users discover, join, or create quests in their local area using an **interactive map-based interface**. Users can explore quests posted by others or set their own, making it a **dynamic and engaging platform for local adventures**.

Whether you’re looking for a fun challenge, exploring your city, or creating experiences for others, Peanut turns everyday locations into exciting quests.

---

## 📌 Features

- **Explore Local Quests** – Browse and join quests created by other users in your vicinity.  
- **Create Your Own Quests** – Set custom quests for friends or the public.  
- **Interactive Map** – Visualize quests and navigate easily with a map interface.  
- **Gamification** – Earn points, badges, and rewards for completing quests.  
- **Community Driven** – Engage with other adventurers, rate quests, and leave feedback.

## 🚀 How It Works

1. **Discover Quests:** Open the app to see nearby quests on the interactive map.  
2. **Join Quests:** Select a quest to view details and accept challenges.  
3. **Complete & Earn:** Finish tasks and gain points, badges, or rewards.  
4. **Create Quests:** Add your own quests for others to explore.
